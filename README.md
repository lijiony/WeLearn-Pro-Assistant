## 🚀 核心功能

WeLearn Pro 专为解决大学英语学习痛点而生，基于最新内核驱动，完美适配 2024-2025 学年课程。

### 1. 全自动课程秒刷 (Auto-Course)
- **智能识别**: 自动获取课程列表，一键解析单元结构。
- **自定义正确率**: 支持 **固定分数** (如100分) 或 **随机分数** (如85-95分)，模拟真实学习轨迹。
- **双通道提交**: 独创 A/B 双通道提交技术，确保数据 100% 上传成功。
- **跳过已完成**: 智能跳过已学完的章节，拒绝重复劳动。

### 2. 智能化时长挂机 (Auto-Time)
- **多线程加速**: 内置 20+ 线程并发处理，刷时长效率提升 300%。
- **灵活设置**: 支持自定义每个练习的挂机时长 (如30秒)，或随机时长模式。
- **防掉线机制**: 独家心跳包技术，保持 Session 活跃，彻底告别掉线烦恼。
- **实时进度**: 可视化进度条，实时掌握挂机状态。

## 🛡️ 安全特性

- **直连模式**: 强制禁用系统代理，无需开启 VPN 即可稳定运行，避免 IP 异常。
- **机器码绑定**: 一机一码授权机制，保障用户权益。
